# 104 Enigmi

Sito statico per generare automaticamente quiz logici in stile TikTok.

## Cosa fa

- Genera quiz sempre diversi con logica procedurale.
- Riduce le ripetizioni salvando gli ultimi quiz generati nel browser.
- Permette di scegliere categoria e difficoltà.
- Mostra soluzione e spiegazione.
- Crea descrizione TikTok pronta con massimo 5 hashtag.
- Genera prompt per copertine verticali 9:16.
- Copia descrizione e prompt con un click.

## Categorie disponibili

- Mix automatico
- Logica
- Matematica
- Attenzione
- Interruttori
- Fiammiferi
- Porte e guardiani
- Bicchieri e tubi

## Come pubblicarlo con GitHub Pages

1. Apri le impostazioni del repository.
2. Vai su Pages.
3. In Build and deployment, scegli Deploy from a branch.
4. Scegli branch main e folder /root.
5. Salva.

Dopo qualche minuto il sito sarà online all'indirizzo GitHub Pages del repository.

## File principali

- index.html: struttura del sito
- style.css: grafica
- script.js: generatore procedurale quiz, descrizioni e prompt
