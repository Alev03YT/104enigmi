# 104 Enigmi

Sito statico per generare automaticamente quiz logici in stile TikTok.

## Cosa fa

- Genera enigmi logici e quiz tranello.
- Mostra la soluzione.
- Crea una descrizione pronta per TikTok.
- Usa massimo 5 hashtag.
- Genera anche un prompt per creare immagini verticali 9:16 stile quiz virale.

## Come pubblicarlo con GitHub Pages

1. Apri le impostazioni del repository.
2. Vai su **Pages**.
3. In **Build and deployment**, scegli:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
4. Salva.

Dopo qualche minuto il sito sarà online all’indirizzo GitHub Pages del repository.

## File principali

- `index.html` struttura del sito
- `style.css` grafica
- `script.js` generatore quiz e descrizioni
